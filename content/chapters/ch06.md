# 第六章：自动化篇 - 让AI每天自动干活

> 这章是整本手册最值钱的部分。设置好之后，你睡觉它也在工作。

## 我现在的自动化清单

每天不用我动手，自动完成的事：

- **09:00** 推送学习简报（摄影、AI工具、创业方法论）
- **10:30** 推送小红书选题策划（3个方向）
- **15:00** 推送下午学习简报
- **21:00** 推送晚间学习简报
- **每晚** 整理当天对话，更新记忆文件

全部推送到飞书，我只需要看就行。

## 怎么设置定时任务（Cron）

在 OpenClaw 控制台（`http://127.0.0.1:18789`）找到「Cron」菜单。

或者直接跟 AI 说：

> "帮我设置一个每天早上9点的任务，搜索宠物摄影相关内容，整理成简报发给我"

AI 会自动创建 cron 任务。

---

## Cron 任务的结构

```json
{
  "name": "每日学习简报",
  "schedule": {
    "kind": "cron",
    "expr": "0 9 * * *",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "搜索今日宠物摄影、AI工具、创业相关内容，整理成简报发到飞书"
  }
}
```

## 心跳机制

心跳是 OpenClaw 定期检查的机制，默认每30分钟触发一次。

你可以在 `HEARTBEAT.md` 里设定心跳要做什么：

```markdown
# HEARTBEAT.md

## 检查清单（每次心跳轮询）
- 有没有重要邮件？
- 有没有2小时内的日程？
- 有没有需要提醒的事？
```

## 实用自动化场景

**场景1：每日学习简报**
搜索你关注的话题，整理成费曼学习法格式，推送到飞书。

**场景2：小红书选题策划**
基于当前热点，给出3个符合你人设的选题方向。

**场景3：晚间复盘提醒**
每晚提醒你填写当天复盘：收入、内容、涨粉、问题、明日计划。

**场景4：记忆维护**
每周自动整理日记，把重要内容升级到 MEMORY.md。

---

## 注意事项

- cron 任务用便宜模型（QwenTurbo），省钱
- 任务不要设太多，每个都会消耗 token
- 先跑一次测试，确认输出符合预期再设定时
