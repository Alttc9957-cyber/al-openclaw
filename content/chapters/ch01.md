# 第一章：安装篇 - 从零安装OpenClaw

> 不会代码也能搞定，跟着做就行。全程大概20分钟。

---

## 第一步：安装 Node.js

OpenClaw 需要 Node.js 才能运行，就像手机需要系统一样。

**Mac 用户：**

1. 打开浏览器，访问 [https://nodejs.org](https://nodejs.org)
2. 点左边绿色按钮（LTS版本）下载
3. 双击安装包，一路点「继续」
4. 打开「终端」（Command+空格，搜索「终端」）
5. 输入以下命令，回车：

```
node -v
```

看到类似 `v20.x.x` 的版本号，说明成功了。

**Windows 用户：**

1. 同样访问 [https://nodejs.org](https://nodejs.org) 下载安装
2. 打开「命令提示符」（Win键，搜索「cmd」）
3. 输入 `node -v` 回车验证

---

## 第二步：安装 OpenClaw

打开终端，复制粘贴这一行命令，回车：

```
npm install -g openclaw
```

等待1-2分钟（会看到一堆滚动的文字，正常的）。

安装完输入：

```
openclaw --version
```

看到版本号就装好了。

---

## 第三步：获取 API Key

OpenClaw 需要连接 AI 模型才能工作。直接用官方的 key 比较贵，推荐用**第三方中转站**，便宜很多。

**我在用的：LTCraft**

1. 访问 [https://ltcraft.ai](https://ltcraft.ai)
2. 注册账号
3. 充值（10块钱能用很久，日常对话每天大概几毛钱）
4. 进入控制台 → 找到「API Key」→ 点复制

---

## 第四步：配置 OpenClaw

在终端输入：

```
openclaw configure
```

按提示填入：

- **API Key**：粘贴刚才复制的 key
- **API Base URL**：`https://api.ltcraft.ai/v1`
- **模型**：`claude-sonnet-4-5`（性价比最高）

---

## 第五步：启动

```
openclaw gateway start
```

看到 `Gateway started` 就成功了。

以后每次开机想用，就运行这一行启动它。

---

## 第六步：连接飞书（推荐）

连上飞书后，AI 助手可以直接在飞书里跟你聊天，还能定时推送消息给你。

1. 访问 [https://open.feishu.cn/app](https://open.feishu.cn/app)
2. 点「创建企业自建应用」
3. 填写应用名称（比如「小助手」）
4. 进入应用 → 「添加应用能力」→ 选「机器人」
5. 在 OpenClaw 控制台配置飞书的 App ID 和 App Secret

详细配置步骤见 [OpenClaw 官方文档](https://docs.openclaw.ai)

---

## 常见问题

**Q：命令跑不了，说「找不到命令」？**

Node.js 没装好，重新装一遍，或者重启终端再试。

**Q：API Key 报错「无效」？**

检查有没有多复制了空格，或者账户余额不足。

**Q：装好了但没反应？**

先运行 `openclaw gateway status` 看状态，再运行 `openclaw gateway start` 启动。
